Sistema operativo Android 10

Android es un sistema operativo para móviles diseñado por la compañía estadounidense Google. Basado en el sistema operativo Linux, su objetivo inicial fue fomentar el uso de un sistema de tipo abierto, gratuito, multiplataforma y muy seguro, adaptado a los dispositivos móviles como smartphones y tablets.

Kernel del Android 10

El sistema android tiene una base en linux la cual es de codigo abierto y esto facilita la manera de hacer cambios en el sistema, el kernel del android 10 es de version 4.19 tiene algunas novedades interesantes entre ellas estan, soporte para el protocolo Wi-Fi 6, viene con mitigaciones para las vulnerabilidades L1 Terminal Fault (L1TF), así como SpectreRSB en el hardware x86, trae además una serie de mejoras para la arquitectura de 64 bits y para el sistema de archivos y también soporte mejorados para gráficos, todo esto permite que android 10 tenga ventajas como un modo oscuro, navegacion por gestos, soportes de redes 5g.

Kernel en versiones anteriores

En el sistema Android 8 Oreo tenia mejoras de seguridad, mejoras de rendimiento de sistema y un soporte para project table que separo la capa hardware de cada sistema, lo que facilitó la actualización del sistema operativo y redujo la fragmentación. En el sistema Android 9 Pie tenia la adaptacion de brillo automatica, gestos de navegacion en vez de botones de navegacion y una gestion de bateria mejorada por mencionar algunos beneficios de las versiones anteriores. En las dos versiones de android se tenia el kernel 4.4

Android 10 y su interfaz de usuario

La interfaz de usuario (IU) de una app para Android se compila como una jerarquía de diseños y widgets. Los diseños son objetos ViewGroup , contenedores que controlan cómo se posicionan sus vistas secundarias en la pantalla. Los widgets son objetos View , componentes de la IU como botones y cuadros de texto.

API en Android

La APIS ayudan a que se puedan conectar las aplciaciones descargadas en el sistema y el software de android para asegurarse de que sea seguro y compatible abrir la aplicacion y hacerla funcionar en el dispositivo y esta misma permite que el mismo sistema demuestre en la interfaz su cambio de color, iluminacion, gestos etc...

Composicion Sistema Operativo Android

Kernel: El núcleo (kernel) de Android 10 es el corazón del sistema operativo. Es el responsable de la gestión de recursos del hardware, como la memoria, el procesador, los controladores de dispositivos y la administración de energía. El kernel es fundamental para el funcionamiento del sistema operativo.

Sistema Android: El sistema Android en sí mismo comprende una variedad de componentes esenciales, como el administrador de ventanas, el administrador de servicios, el administrador de paquetes y otros servicios del sistema. Estos componentes son responsables de administrar las aplicaciones y proporcionar una interfaz de usuario coherente.

Biblioteca de tiempo de ejecución de Android (ART): Android 10 utiliza la máquina virtual ART para ejecutar aplicaciones. ART compila el código de las aplicaciones en tiempo de instalación, lo que mejora el rendimiento y la eficiencia en comparación con la máquina virtual Dalvik utilizada en versiones anteriores de Android.

Framework de Android: El framework de Android 10 proporciona una variedad de bibliotecas y servicios que los desarrolladores utilizan para crear aplicaciones. Incluye API para gráficos, administración de ventanas, comunicación de red, administración de bases de datos, acceso a sensores y más.

Aplicaciones del sistema: Android 10 incluye una serie de aplicaciones del sistema, como el marcador, el correo electrónico, el navegador, el calendario, la aplicación de cámara y otras aplicaciones preinstaladas. Estas aplicaciones se integran estrechamente con el sistema operativo y proporcionan funcionalidades esenciales para los usuarios.

Motor de renderizado: Android 10 utiliza un motor de renderizado para representar la interfaz de usuario en la pantalla. En versiones anteriores, Android utilizaba WebView para el motor de renderizado, pero en Android 10, se introdujo un motor de renderizado llamado "Grafika" para mejorar el rendimiento y la seguridad.

Seguridad: Android 10 incluye una serie de características de seguridad, como el sistema de permisos granulares, el sistema de autenticación biométrica, la prevención de acceso no autorizado y la encriptación de datos en el dispositivo.

Actualizaciones de seguridad y del sistema: Android 10 introduce actualizaciones de seguridad y del sistema a través de Google Play Store, lo que permite a los dispositivos recibir correcciones de seguridad y mejoras de manera más eficiente sin necesidad de actualizaciones completas del sistema operativo.

Gestión de energía: Android 10 incluye características de administración de energía para prolongar la duración de la batería, como el modo de ahorro de energía, la gestión de aplicaciones en segundo plano y la optimización de la batería.

Núcleo de comunicaciones: Android 10 incluye soporte para tecnologías de comunicación como Wi-Fi, Bluetooth, NFC y telefonía móvil, lo que permite a los dispositivos Android conectarse a redes y otros dispositivos.
